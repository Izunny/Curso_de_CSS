Tabla responsiva

Una tabla adaptable mostrará una barra de desplazamiento horizontal si la pantalla es demasiado pequeña para mostrar el contenido completo. Cambie el tamaño de la ventana del navegador para ver el efecto:

Para crear una tabla responsiva, agregue un elemento contenedor (como div) con overflow-x:auto alrededor del elemento de tabla:
